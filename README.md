# Quasar

Quasar is meant to take the hassle out of uploading your data to Amazon Redshift.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quasar'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quasar
